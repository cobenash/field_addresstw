<h2>EN</h2>
The is a field which provided the Address function in Taiwan. Integrate with <a href="https://github.com/essoduke/jQuery-TWzipcode">jQuery-TWzipcode</a> Plugin. it's light and easy to use.

<h2>Features</h2>
1. Provide County/Distrinct/Zipcode/Address for Taiwan and developed on javascript.
2. Provide two formatter. Full Taiwan Address Text and Simple Taiwan Address Text.

<h2>Similar Modules</h2>
1. <a href="https://www.drupal.org/sandbox/gloomcheng/1872126">Address Field Taiwan</a>.
Unlike Address Field Taiwan, this module does not integrate the <a href="https://www.drupal.org/project/addressfield">Addressfield</a> module, which can be used independently and lightweight.

<h2>Problems need to be fix</h2>
1. Field required is not working.
2. not support token.
3. The data can't be changed or modify because the data isn't save at Drupal Database.


<h2>中文</h2>
這個模組提供Taiwan地址的Field。簡單整合<a href="https://github.com/essoduke/jQuery-TWzipcode">jQuery-TWzipcode</a>的Jquery插件，介面簡單且輕巧。

<h2>功能介紹</h2>
1. 提供 縣市/區/郵遞區號 與地址欄位，並且使用Javascript整合。
2. 提供兩種Formatter樣式（Full Taiwan Address Text）與（Simple Taiwan Address Text）。

<h2>類似模組</h2>
1. <a href="https://www.drupal.org/sandbox/gloomcheng/1872126">Address Field Taiwan</a>
不同於 Address Field Taiwan，這個模組並非整合<a href="https://www.drupal.org/project/addressfield">Addressfield</a>模組，這個模組可以獨立使用且輕巧。

<h2>待改進問題</h2>
1. Field欄位無法改成必填，需要修正。
2. 可能不支援Token。
3. 縣市區域與郵遞區號的資料無法從模組中更改，因為資料並非存在Drupal的資料庫之中。
